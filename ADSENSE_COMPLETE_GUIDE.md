## 📊 AdSense 성과 모니터링

### 1. AdSense 대시보드 지표

**일일 확인 항목:**
- 예상 수익
- 페이지 조회수
- 페이지 CTR
- 페이지 RPM (1,000회 노출당 수익)

**주간 분석:**
- 가장 수익성 높은 광고 단위
- 가장 수익성 높은 페이지/탭
- CTR 트렌드
- CPC 트렌드

### 2. Google Analytics 연동

AdSense와 Analytics를 연결하여:
- 어떤 트래픽 소스가 가장 수익성이 높은지 파악
- 사용자 행동과 광고 클릭 상관관계 분석
- A/B 테스트로 광고 위치 최적화

### 3. 최적화 전략

**CTR 개선:**
- 광고 위치 조정
- 광고 크기 실험
- 광고 형식 변경 (텍스트/디스플레이/네이티브)

**수익 증대:**
- 고CPC 키워드 콘텐츠 추가
- 트래픽 증가 전략 (SEO, 소셜, 광고)
- 사용자 체류 시간 증가

## ⚠️ AdSense 정책 준수

### 금지 사항

1. **자가 클릭 금지**
   - 본인의 광고 절대 클릭 금지
   - 테스트용 클릭도 금지
   - VPN/시크릿 모드로 클릭해도 감지됨

2. **클릭 유도 금지**
   - "광고를 클릭해주세요" 문구 금지
   - 광고 주변에 화살표/강조 금지
   - 오해를 일으키는 광고 배치 금지

3. **무효 트래픽 금지**
   - 봇 트래픽
   - 자동화된 트래픽
   - 클릭 교환 프로그램

### 위반 시 제재

- 1차 경고: 광고 제한 (일시 중단)
- 2차 경고: 계정 정지 (최대 30일)
- 3차 경고: 계정 영구 폐쇄

### 정책 준수 팁

✅ **안전한 방법:**
- 자연스러운 콘텐츠 배치
- 양질의 트래픽 확보
- 정기적인 정책 검토
- 의심스러운 활동 즉시 보고

## 🆘 문제 해결

### "광고가 표시되지 않아요"

**원인 1: 사이트 미승인**
- 해결: AdSense 콘솔에서 승인 상태 확인
- 대기 시간: 1-2주

**원인 2: 광고 차단기**
- 해결: 시크릿 모드에서 테스트
- 광고 차단기 비활성화

**원인 3: 광고 단위 ID 오류**
- 해결: slot ID가 올바른지 확인
- AdSense 콘솔에서 광고 단위 ID 재확인

### "수익이 예상보다 낮아요"

**원인 1: 트래픽 부족**
- 해결: SEO 최적화, 마케팅 강화

**원인 2: CTR 낮음**
- 해결: 광고 위치 조정, A/B 테스트

**원인 3: CPC 낮음**
- 해결: 고가치 키워드 콘텐츠 추가
