# Google AdSense 설정 가이드

## 1. Google AdSense 계정 생성

### 1.1 AdSense 가입
1. [Google AdSense](https://www.google.com/adsense)에 접속
2. **시작하기** 클릭
3. Google 계정으로 로그인
4. 웹사이트 URL 입력
5. 이메일 수신 동의 여부 선택
6. **AdSense 시작하기** 클릭

### 1.2 사이트 연결
1. **사이트 추가** 클릭
2. 도메인 입력 (예: `https://your-domain.com`)
3. **AdSense 코드 복사** (나중에 사용)
4. 사이트 승인 대기 (보통 1~2주 소요)

---

## 2. 퍼블리셔 ID 및 광고 단위 설정

### 2.1 퍼블리셔 ID 확인
1. AdSense 대시보드 접속
2. **계정** → **계정 정보** 클릭
3. **퍼블리셔 ID** 복사 (형식: `ca-pub-XXXXXXXXXXXXXXXX`)

### 2.2 광고 단위 생성
각 광고 위치마다 별도의 광고 단위를 생성하는 것이 좋습니다.

**광고 단위 1: 비교 결과 하단**
1. **광고** → **개요** → **광고 단위 기준** 클릭
2. **디스플레이 광고** 선택
3. 이름: "비교결과_하단" 입력
4. 광고 크기: **반응형** 선택
5. **만들기** → **광고 단위 ID** 복사 (예: `1234567890`)

**광고 단위 2: 멀티 통화 탭**
1. 이름: "멀티통화_하단"
2. 광고 크기: **반응형**
3. 광고 단위 ID 복사 (예: `0987654321`)

**광고 단위 3: 히스토리 탭**
1. 이름: "히스토리_하단"
2. 광고 크기: **반응형**
3. 광고 단위 ID 복사 (예: `1122334455`)

## 3. 코드에 ID 적용

### 3.1 index.html 수정
`index.html` 파일에서 퍼블리셔 ID 교체:

```html
<!-- 변경 전 -->
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
        crossorigin="anonymous"></script>

<!-- 변경 후 -->
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-1234567890123456"
        crossorigin="anonymous"></script>
```

### 3.2 App.jsx 광고 단위 ID 수정

**AdSenseAd 컴포넌트에서 퍼블리셔 ID 교체:**
```javascript
// /src/App.jsx 파일 내 AdSenseAd 컴포넌트
data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"  // 여기를 실제 ID로 변경
```

**각 광고 위치의 slot ID 교체:**

1. **비교 결과 하단 광고** (약 870번째 줄):
```javascript
<AdSenseAd
  slot="1234567890"  // 실제 광고 단위 ID로 교체
  format="auto"
  responsive={true}
  style={{ minHeight: 90 }}
/>
```

2. **멀티 통화 탭 광고** (약 1075번째 줄):
```javascript
<AdSenseAd
  slot="0987654321"  // 실제 광고 단위 ID로 교체
  format="auto"
  responsive={true}
  style={{ minHeight: 90 }}
/>
```

3. **히스토리 탭 광고** (약 963번째 줄):
```javascript
<AdSenseAd
  slot="1122334455"  // 실제 광고 단위 ID로 교체
  format="auto"
  responsive={true}
  style={{ minHeight: 90 }}
/>
```

## 4. 광고 배치 전략

### 현재 광고 위치

**위치 1: 비교 결과 테이블 하단**
- **타이밍**: 사용자가 비교 결과를 확인한 직후
- **이유**: 가장 높은 참여도, 정보 탐색 완료 시점
- **예상 CTR**: 높음 (사용자가 다음 행동을 고민하는 시점)

**위치 2: 멀티 통화 비교 탭 하단**
- **타이밍**: 여러 통화 비교 차트를 본 후
- **이유**: 심화 정보 확인 후 쉬는 시점
- **예상 CTR**: 중간

**위치 3: 히스토리 탭 하단**
- **타이밍**: 환율 히스토리 차트를 본 후
- **이유**: 데이터 분석 완료 시점
- **예상 CTR**: 중간

### 추가 광고 위치 제안

현재 구현되지 않았지만, 향후 추가 가능한 위치:

**위치 4: 페이지 상단 (헤더 아래)**
- 코드:
```javascript
<AdSenseAd
  slot="YOUR_SLOT_ID"
  format="horizontal"
  responsive={true}
  style={{ minHeight: 60, marginBottom: 16 }}
/>
```

**위치 5: 사이드바 (데스크톱 전용)**
- 큰 화면에서만 표시되는 세로형 광고
- CSS media query로 모바일에서는 숨김 처리

## 5. 광고 정책 준수

### AdSense 정책
- ❌ **클릭 유도 금지**: "광고를 클릭하세요" 등의 문구 금지
- ❌ **광고 레이블 변경 금지**: "광고" 레이블을 "스폰서" 등으로 변경 불가
- ❌ **허위 클릭 금지**: 자신의 광고를 클릭하거나 클릭 유도 금지
- ✅ **가치 있는 콘텐츠**: 실제 가치가 있는 정보 제공
- ✅ **사용자 경험 우선**: 광고가 콘텐츠를 방해하지 않도록

### 금융 콘텐츠 특별 규정
해외 송금 비교 서비스는 금융 관련 콘텐츠이므로:
- 정확한 정보 제공 필수
- 면책 조항 명시 (이미 구현됨)
- 특정 서비스 과도한 추천 금지 (이미 구현됨)

## 6. 수익 최적화 팁

### 6.1 광고 배치 실험
AdSense의 **실험** 기능을 활용하여 A/B 테스트:
- 광고 위치 변경
- 광고 크기 조정
- 광고 스타일 변경

### 6.2 고수익 키워드 타겟팅
현재 콘텐츠가 이미 타겟팅하고 있는 고수익 키워드:
- "해외송금" - CPC 높음
- "환율" - CPC 중간
- "송금 수수료" - CPC 높음
- "실시간 환율" - CPC 중간

추가 키워드 제안:
- "외화 송금"
- "국제 송금"
- "해외 송금 비교"
- "환전 수수료"

### 6.3 트래픽 증가 전략
더 많은 광고 수익을 위해:
- SEO 최적화 (meta tags, structured data)
- 소셜 미디어 공유 버튼 추가
- 블로그 콘텐츠 추가 (환율 분석, 송금 가이드)
- 정기적인 데이터 업데이트 (신뢰도 향상)

## 7. 예상 수익 계산

### 수익 계산식
```
월 수익 = (월 페이지뷰 × 광고 노출률) × CTR × CPC
```

### 예시 시나리오

**시나리오 1: 초기 단계**
- 월 페이지뷰: 10,000
- 광고 노출률: 80% (8,000회)
- CTR: 0.5% (40 클릭)
- CPC: ₩500
- **예상 월 수익: ₩20,000**

**시나리오 2: 성장 단계**
- 월 페이지뷰: 50,000
- 광고 노출률: 80% (40,000회)
- CTR: 0.8% (320 클릭)
- CPC: ₩600
- **예상 월 수익: ₩192,000**

**시나리오 3: 성숙 단계**
- 월 페이지뷰: 200,000
- 광고 노출률: 85% (170,000회)
- CTR: 1.0% (1,700 클릭)
- CPC: ₩700
- **예상 월 수익: ₩1,190,000**

*실제 수익은 광고주 경쟁, 시즌, 사용자 위치 등에 따라 달라집니다.*

## 8. 성과 모니터링

### AdSense 대시보드 지표
정기적으로 확인할 지표:
- **페이지 RPM**: 페이지뷰 1,000회당 수익
- **CPC**: 클릭당 비용
- **CTR**: 클릭률
- **노출수**: 광고 표시 횟수

### GA4와 연동 분석
Google Analytics 4와 함께 사용하여:
- 어떤 탭에서 광고 클릭이 많은지
- 어떤 통화 비교 시 참여도가 높은지
- 사용자 체류 시간과 광고 노출의 상관관계

## 9. 문제 해결

### 광고가 표시되지 않는 경우

**1. 코드 확인**
```javascript
// 브라우저 콘솔에서 실행
console.log(window.adsbygoogle);
// undefined가 아니라면 스크립트 로드 성공
```

**2. 광고 차단기 확인**
- 개발 중에는 광고 차단기를 비활성화
- 시크릿 모드에서 테스트

**3. 승인 대기 중**
- 새 사이트는 승인까지 1~2주 소요
- 승인 전에는 빈 공간만 표시됨

**4. 트래픽 부족**
- 일일 페이지뷰 100 미만이면 광고가 제한될 수 있음
- 트래픽이 증가하면 자동으로 활성화

### 수익이 예상보다 낮은 경우
- 광고 위치 재조정
- 콘텐츠 품질 개선
- 고수익 키워드 추가
- 사용자 경험 개선으로 체류 시간 증가

## 10. 대안 광고 플랫폼

Google AdSense 외에 추가 수익원:

### 제휴 마케팅
- 송금 서비스 제휴 프로그램
- Wise, SentBe 등의 추천 링크
- 수수료: 사용자당 ₩5,000~₩20,000

### Carbon Ads
- 기술/금융 콘텐츠에 특화
- 깔끔한 디자인
- 고정 CPM

### 직접 광고 판매
- 트래픽이 충분히 증가하면
- 송금 서비스 직접 접촉
- 배너 광고 판매
